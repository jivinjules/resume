Built with HTML, CSS, JavaScript, jquery, Firebase, Google Fonts, and moment.js, Train Scheduler allows the user to operate a fictional train station. Trains and their respective arrival times are stored in Firebase, and moment.js controls the timing.

Built with HTML, CSS, and JavaScript, Hangman randomly generates a word, converts it to blanks, and provides the user 10 guesses. If correct, the user will be provided with an image of the word.

What's for Dinner, DFW? is a collaborative effort of four developers. The site uses Zomato API, Yummly API, and Google Maps API make dinnertime easy for residents of DFW. The user can choose a cuisine and a city and is offered recipes to make (if they decide to eat at home) or restaurants which offer that cuisine (if they decide to eat out), and Google Maps shows the exact location of the restaurant.

Crystal counter is an interactive game where the a random number is provided, and four crystal pictures are also given random values. The user must then try to match the random number by clicking on the crystals. Built with HTML, CSS, JavaScript, and jquery, the site provides a new number and values for each game.

The Periodic Table Trivia Game is a timed, multiple choice trivia game. Built with jquery timers, the user has 15 seconds to correctly choose an answer. The game keeps track of the score and also provides the correct answer and image.

Play the Trivia Game!

LIRI (aka Language Interpretation and Recognition Interface) is a command line node app that takes in parameters and gives you back data (like SIRI but for language only). LIRI is built with npm and the following API's (bandsintown, OMDB, Spotify, and moment) and can return information about concerts, movies, and music.

Check out LIRI!

Bamazon is an Amazon-type store front with three options: Customer, Manager, or Supervisor which runs with Node.js and MySQL backend. The customer is able to view and 'purchase items' by department. The manager is able to view/add inventory, and the Sueprvisor is able to sort sales by department or add a new department.

Check out Bamazon!
